Project Overview
This project is part of a Data Science course (Milestone 2). It explores a dataset of student performance in secondary education, aiming to predict the final grade (**G3**) using various demographic and social features.
 Dataset Description
The dataset student_data.csv includes student grades, demographic, social, and school-related features. 
- Target Variable: G3 (Final Grade).
- Features: Attributes like study time, failures, family support, absences, etc.
In this lab, we completed the following:
1. Data Cleaning: Handled missing values and ensured all column names were properly formatted.
2. Exploratory Data Analysis (EDA): Created visualizations to understand the distribution of grades and the impact of factors like absences.
3. Data Preprocessing: Converted categorical variables into numerical format using One-Hot Encoding.
4. Machine Learning Modeling: - Trained a Decision Tree Classifier.
   - Trained a Random Forest Classifier.
5. Evaluation: Compared the accuracy of both models to find the most effective predictor.
